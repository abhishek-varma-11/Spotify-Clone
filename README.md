# Spotify Clone 🎵

A hand-coded Spotify-inspired web application built as part of my web development learning journey.

## Live Demo

🔗 **GitHub Pages:**  https://abhishek-varma-11.github.io/Spotify-Clone/

---

## About The Project

This project was created to strengthen my understanding of front-end web development by recreating the user interface of a popular music streaming platform.

The goal was not to build a production-ready application, but to apply and practice concepts learned while studying HTML, CSS, UI design, and Git. Every section of the application was built manually without using website builders or pre-made templates.

Throughout the development process, I focused on understanding how modern web interfaces are structured, styled, and animated while gaining hands-on experience solving real-world layout and design challenges.

This project represents my current stage of learning, and I will continue refining and expanding it as my skills grow.

## Challenges Faced and Lessons Learned

Working on this project helped me gain practical experience with several important front-end development concepts.

Some of the areas I explored include:

* Flexbox layouts and positioning
* Relative and absolute positioning
* Overflow management
* Text truncation using ellipsis
* SVG integration and styling
* CSS animations and transitions
* Hover interactions
* Scroll behavior and custom UI controls
* Responsive sizing units such as `%`, `rem`, `vw`, and `vh`
* Organizing larger CSS codebases
* Git and GitHub workflows

Many features required experimentation, debugging, and research before reaching a working solution. Solving these challenges helped me better understand how modern web applications are built and styled.
